# PhrasingPlus

PhrasingPlus is extension for Phrasing. It's a mountable Rails engine used for inline editing of images.

## Installation

### Add to Gemfile

```ruby
gem 'phrasing_plus'
```

### Copy migrations

```shell
rake phrasing_plus:install:migrations
```

### Run migrations

```shell
rake db:migrate
```
